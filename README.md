## Synopsis

The following program takes as input an integer and outputs
whether or not the integer is a prime number.

## Example

gcc -o prime prime.c

./prime.exe
Enter a number: 6
6 is NOT prime.

./prime.exe
Enter a number: 5
5 is prime.
